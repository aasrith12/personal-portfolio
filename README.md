# Personal Portfolio

A responsive portfolio website built with React and Vite, showcasing experience, projects, education, and a detailed About page.

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- React Icons

## Features

- Hero section with profile card and social links
- Skills marquee with hover-to-pause behavior
- Work experience timeline
- Projects section with GitHub links
- Education section with institution logos
- About page with:
  - Personal story
  - Slideshow highlights
  - Interests and hobbies
- Resume button opening PDF in a new tab
- Shared footer across pages

## Local Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
